The folder names are in all caps with any necessary information below them

DOC:
This folder contains all the Javadoc files for the project. To access it, open index.html in the 
root of the doc folder.

OTHER DOCS:
On GitHub link, see the commits page for a list of all changes made to the software throughout.

changed, why it was changed, problems in that version, etc and also the lines of code that were 
actually changed in the project.
